# Exploratory Data Analysis Project

*Project Overview*
This project performs Exploratory Data Analysis (EDA) on a cleaned layoffs dataset.

*Objectives*
- Analyze layoff trends
- Identify top companies
- Study industry impact

*Key Insights*
- Highest layoffs in tech industry
- Peak layoffs observed in 2022
- Some companies had 100% layoffs

*Tools Used*
- MYSQL Workbench
